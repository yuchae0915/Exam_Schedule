# 考試時程表系統

簡潔高效的考試進度追蹤與時程管理工具

## 🚀 快速開始

### 安裝（首次使用）

1. **安裝 Node.js**
   - 下載：https://nodejs.org/（選擇 LTS 版本）
   - 安裝完成後，在專案資料夾開啟命令提示字元
   
2. **安裝依賴**
   ```bash
   npm install
   ```

### 使用

- **啟動**：雙擊 `open.bat`（自動啟動伺服器並開啟瀏覽器）
- **訪問**：http://localhost:8000
- **關閉**：關閉所有網頁標籤，伺服器會在 10 秒後自動停止

## ✨ 功能特色

- 🔄 **自動關閉**：關閉所有網頁後 10 秒自動停止伺服器
- 💾 **手動備份**：透過網頁界面手動備份和恢復數據
- 🎯 **背景運行**：伺服器在背景執行，無擋路視窗
- 📊 **進度追蹤**：考試倒數、科目進度、讀書計畫管理

## 📂 專案結構

```
├── frontend/           # 前端檔案
│   ├── script/        # JavaScript 腳本
│   └── style/         # CSS 樣式
├── server.js          # Node.js 伺服器
├── open.bat           # 一鍵啟動腳本
└── package.json       # 依賴設定
```

## 💾 數據備份

在任何頁面點擊底部的「💾 備份數據」按鈕即可：
- **備份**：下載 JSON 檔案保存所有數據
- **恢復**：上傳備份檔案恢復數據

## 🛠️ 開發說明

- 修改前端檔案（HTML/CSS/JS）後，按 F5 刷新即可
- 修改 `server.js` 後，需要重啟伺服器（關閉所有網頁，重新雙擊 `open.bat`）
