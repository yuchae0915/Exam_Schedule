# 考試時程表 📚

一個用於管理和追蹤公職考試時程的網頁應用程式。

## ✨ 功能特色

- 📅 考試倒數計時器
- 📊 考試科目與權重顯示
- 💾 數據備份與恢復功能
- 📱 響應式設計，支援手機瀏覽
- 🎯 完全離線運作，無需網路連線

## 🚀 使用方式

### 快速啟動

雙擊 `open.bat` 即可在瀏覽器中開啟應用。

### 數據備份

1. 點擊「💾 備份數據」按鈕
2. 系統會自動下載一個 JSON 檔案
3. 妥善保存此檔案

### 數據恢復

1. 點擊「📥 恢復數據」按鈕
2. 選擇之前備份的 JSON 檔案
3. 確認後系統會清空現有數據並恢復備份

⚠️ **注意**：恢復數據會完全替換目前的所有記錄，此操作無法撤銷！

## 📁 檔案結構

```
Exam_Schedule/
├── frontend/           # 網頁文件
│   ├── index.html     # 主頁面
│   ├── style/         # 樣式檔
│   └── script/        # JavaScript 檔案
└── open.bat           # 啟動器
```

## 💡 技術說明

- **純靜態網頁**：不需要伺服器，直接用瀏覽器開啟即可
- **本地儲存**：使用瀏覽器的 localStorage 儲存所有數據
- **離線優先**：完全不需要網路連線

## 📝 注意事項

1. 數據儲存在瀏覽器中，請定期備份
2. 清除瀏覽器數據會導致資料遺失
3. 建議定期使用備份功能保存進度
4. 備份檔案包含時間戳，方便識別版本

## 🔧 維護

如需清理臨時檔案，可執行 `cleanup.bat`
