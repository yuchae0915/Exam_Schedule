考試時程表 - 純靜態網頁版本
===================================

📌 架構說明
-----------
本應用現在是一個完全獨立的靜態網頁，不需要任何伺服器或後端支援。

✨ 主要特點
-----------
✅ 無需安裝 Node.js
✅ 無需啟動伺服器
✅ 雙擊即可開啟
✅ 完全離線運作
✅ 數據儲存在瀏覽器

🚀 使用方式
-----------
1. 雙擊 open.bat
2. 瀏覽器會自動開啟 index.html
3. 開始使用！

💾 數據管理
-----------
- 數據儲存：瀏覽器 localStorage
- 備份方式：點擊「備份數據」下載 JSON 檔
- 恢復方式：點擊「恢復數據」上傳 JSON 檔
- 重要提醒：定期備份，避免瀏覽器清除數據導致遺失

📁 核心文件
-----------
frontend/
  ├── index.html              # 主頁面
  ├── countdown.html          # 倒數計時頁
  ├── gaokao_115.html        # 高考計劃頁
  ├── 114_state_exam.html    # 國營考試頁
  ├── style/                 # 樣式檔案
  └── script/
      ├── index.js           # 主要邏輯
      └── backup.js          # 備份功能

🗑️ 已移除的文件
--------------
以下文件已不再需要（但保留在目錄中供參考）：
- server.js               # Node.js 伺服器
- start_hidden.vbs        # 隱藏啟動腳本
- heartbeat.js           # 心跳檢測
- package.json           # npm 配置
- node_modules/          # Node.js 依賴

📝 技術細節
-----------
- 前端框架：純 HTML/CSS/JavaScript
- 數據儲存：localStorage API
- 備份格式：JSON
- 相容性：所有現代瀏覽器

⚠️ 注意事項
-----------
1. 請勿清除瀏覽器數據，否則會遺失所有記錄
2. 建議每週進行一次備份
3. 備份檔案請妥善保存
4. 使用相同瀏覽器才能讀取相同的數據
